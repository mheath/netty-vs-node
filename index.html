What are Netty and Node.js?

Node.js

  • Node is the V8 JavaScript engine outside of the browser
  • Has additional APIs for doing network and file I/O
  • Most commonly used for building asynchronous event-driven network applications

Netty

  • Red Hat project for building event-driven network applications
  • Runs on the Java platform

What does it mean to be event-driven?

Thread-per-connection Architecture

  • Each network connection is processed by its own thread
  • This is how most servers work today

Thread-per-connection Pseudo Code Echo Server

ServerSocket serverSocket = new ServerSocket(8080);

while (true) {

Socket socket = serverSocket.accept();

new Thread() {

byte[] buffer = socket.read();

socket.write(buffer);

}.start();

}

Event-driven

In an event-driven architecture, a single thread can handle multiple network connections.

Event-driven Architecture Myths

  • Event-driven servers are always faster
  • Thread-per-connection servers can't scale
